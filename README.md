# NIKKE 红球箱子策略模拟器

这是一个可直接部署到 GitHub Pages 的静态网页版本。

## 文件结构

- `index.html`
- `styles.css`
- `app.js`

## 本地打开

直接双击 `index.html`，或用任意静态服务器打开。

## 部署到 GitHub Pages

1. 把这三个文件提交到仓库根目录。
2. 在 GitHub 仓库的 `Settings -> Pages` 中选择部署分支。
3. 如果使用根目录部署，页面入口就是 `index.html`。

## 当前特性

- 单页自上而下布局
- 主线 / 活动 / 额外来源 / 策略可直接编辑
- 自动计算每日小时数
- 策略对比
- 鼠标跟随 tooltip
- 导出当前明细 CSV
- 导出图表 PNG
